### 项目简介
这是一个修仙题材的Godot游戏项目，采用C#开发，包含大地图、战斗系统、修炼系统、小游戏等多个模块。

### 快速开始
1. 打开项目前，请完整阅读 `代码规范.md`
2. 所有代码修改必须遵循既定的命名和结构规范
3. 使用模块化设计，确保各模块独立可测试
